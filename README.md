# a GNN framework for genomics
This is a python package for genomics study with a GNN(Graph Neural Networks) framework.

# Getting started

## Prerequisite
+ cython
+ numpy
+ Biopython
+ pytorch 1.9.0
+ pytorch\_geometric 1.7.0

## Install
```shell
git clone https://github.com/deepomicslab/GNNFrame.git
```
